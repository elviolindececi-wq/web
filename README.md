# El violín de Ceci — sitio listo para GitHub Pages

## Qué incluye
- Landing principal premium en `/`
- Test para novios en `/test/`
- Reseñas en `/reviews/`
- Estilo responsive y listo para publicar como sitio estático

## Publicar en GitHub Pages
1. Subí todo el contenido de esta carpeta a un repositorio.
2. En GitHub, abrí **Settings > Pages**.
3. En **Build and deployment**, elegí **Deploy from a branch**.
4. Seleccioná la rama principal y la carpeta `/root`.
5. Guardá.

## Personalizaciones recomendadas
- Reemplazar los bloques visuales del home por fotos reales.
- Cambiar los textos de testimonios por testimonios reales.
- Mantener los enlaces de Calendly, Instagram y WhatsApp si ya son los definitivos.

## Estructura
- `index.html` → home
- `style.css` → estilos del home
- `app.js` → menú y animaciones del home
- `assets/` → SVGs decorativos
- `test/` → test para novios original
- `reviews/` → módulo de reseñas original
